# QwickApps React Framework Documentation

A complete React framework for building modern, responsive applications with intelligent navigation, flexible layouts, and a comprehensive theming system.

## Features
- Responsive navigation for mobile, tablet, and desktop
- Flexible layout system: hero blocks, multi-column layouts, content blocks, feature grids, section containers
- Advanced theming: light/dark/system themes, built-in and custom palettes, CSS variables
- Component library: logo, theme controls, form components, safe components
- Developer experience: TypeScript, Storybook, performance, accessibility

## Installation
```bash
npm install @qwickapps/react-framework
```
